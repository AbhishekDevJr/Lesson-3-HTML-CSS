@Inline Elements Vs Block Level Elements!

@Inline Elements

An inline element does not take up the whole width of the page and multiple inline elements can be used in the same line.\ For Example @Small is an inline element, multiple small elements can be used in the same line.

@Block Level elements

A block level element takes up the whole line and takes the full width of the page.

Elements like Headers and Divs are all block level elements.
